Native Extension Features

A comprehensive overview of every feature built directly into Tabvora. No fluff, no remote dependencies.

Core Architecture

Contextual Workspace Dashboards

Group active browser windows into isolated projects (e.g., Development, Research, Finance). Switch contexts instantly without losing tab state or cluttering your screen.

Performance

Native Memory Hibernation

Automatically discards inactive tab memory footprints using Chrome native discard APIs without closing your tabs or losing navigation state.

Utility

Local Workspace Backup & Export

Export and import entire workspace structures, layout configurations, and saved tab groups as standardized local JSON files.

Navigation

Fuzzy Search & Command Palette

Execute instant searching across open tabs, archived sessions, and workspace titles directly via standard keyboard shortcuts.

Recovery

Local Draft Recovery Options

An optional local feature that preserves crash state and closed session snapshots strictly in your local browser IndexedDB storage.

Privacy

Local Analytics Module

An optional local-only metrics view detailing total tabs closed, memory saved, and workspace usage. Data is processed locally and never transmitted.
