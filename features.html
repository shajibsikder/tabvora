Native Extension Features

A comprehensive overview of every feature built directly into Tabvora. No fluff, no remote dependencies.

Core Architecture

Workspace Shortcut Folders

Organize saved shortcuts into custom, color-coded workspace folders (e.g., Development, Research, Finance) on your new-tab dashboard. Switch between them instantly.

Performance

Virtual Scrolling Grid

Only the shortcuts currently in view are rendered, so the dashboard stays fast and responsive even with hundreds of saved shortcuts.

Utility

Local Backup & Export

Export and import your shortcuts, workspace folders, and settings as a standardized local JSON backup file — no account or cloud required.

Navigation

Deep Shortcut Search

Instantly search your saved shortcuts by title, URL, domain, or notes, with configurable search scope and keyboard-first navigation.

Recovery

Local Draft Recovery (Optional, Default OFF)

When enabled, temporarily records unsaved text-field input across tabs so you don't lose it after an accidental close. Snapshots stay in local IndexedDB and password/payment fields are automatically excluded.

Privacy

Website Usage Statistics (Optional, Default OFF)

An optional, local-only view of domain visit frequency and time spent, rendered as charts. Data is calculated and stored strictly on your device and never transmitted.
