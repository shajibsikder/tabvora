Privacy Policy

Last Updated: January 15, 2026 • Version 1.0

1. Core Privacy Philosophy: Local First

Tabvora is architected from the ground up to respect user privacy completely. We believe that your browsing context, workspace structures, open tabs, and daily activity belong exclusively to you. Tabvora does not contain telemetry, tracking code, analytics scripts, or remote server dependencies.

2. Data Collection Statement

We collect zero personal data.

  • No Analytics: We do not track page views, clicks, or extension lifecycle events.
  • No Telemetry: No error reports or usage metrics are transmitted across the network.
  • No Remote Code: In accordance with Chrome Web Store Manifest V3 requirements, all executable code is packaged locally inside the extension archive.

3. Optional Local Features

Tabvora includes optional productivity modules (such as Local Draft Recovery and Workspace Usage Statistics). These modules are OFF by default and operate purely on local storage (Chrome IndexedDB). Enabling these options will never cause data to leave your physical hardware.

4. How Data is Stored and Erased

All workspace names, tab configurations, and layout settings are stored locally via standard browser storage APIs (chrome.storage.local and IndexedDB). Uninstalling the Tabvora extension instantly and permanently wipes all stored workspace data from your computer.

5. Contact and Policy Updates

If we update this privacy document, changes will be published directly to this page alongside clear release notes in our Changelog.
