Product Roadmap

Our transparent release timeline. Built with zero tracking, local-first storage, and strict Google Developer Policy compliance.

Completed

v1.0.0 – Production Baseline

July 2026

Full Manifest V3 architecture, local-first workspace manager, Scratchpad widget, and custom theme launcher.

  • ✓ Strict Manifest V3 Service Worker background engine
  • ✓ Zero remote network calls, tracking pixels, or telemetry
  • ✓ IndexedDB local database layer for instant performance
In Progress

v1.1.0 – Local Backup & Advanced Widgets

Q4 2026

Enhancing data portability and dashboard customizability without introducing any third-party cloud connections.

  • • Encrypted JSON Export/Import: Backup your shortcuts and workspace structures directly to your local file system.
  • • Custom Wallpaper Engine: Support for local custom image upload and gradient presets for the dashboard background.
  • • Keyboard Shortcut Manager: Extended global hotkey bindings for opening workspace folders instantly.
Planned

v1.2.0 – Smart Focus Tools

Q1 2027

Focus mode integrations designed to reduce screen clutter and improve browser memory efficiency.

  • • Tab Group Auto-Organizer: Automatic grouping of open tabs by host domain using Chrome Tab Groups API.
  • • Focus Timer Widget: Local Pomodoro timer card on the dashboard to help manage work cycles.
