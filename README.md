# Tabvora Official Web Architecture

This directory contains the source code for the official website of **Tabvora – Smart Tab & Workspace Dashboard**.

## Deployment
The project is built with zero package managers or compilation pipelines. It runs directly on GitHub Pages.

1. Push all files to the `main` branch.
2. Enable **GitHub Pages** in your repository settings selecting `/ (root)`.
3. The site will be instantly live at your GitHub Pages address.
