Version Release Notes

Detailed revision history and security compliance log for Tabvora.

v1.0.0

Initial Public Candidate

July 23, 2026

This is the first stable release of Tabvora architected for the Google Chrome Web Store under the strict Manifest V3 specification.

New Features & Architecture

  • • Local Workspace Dashboard: Added custom grid launcher, categories, theme customization, and workspace management.
  • • Local Scratchpad Widget: Integrated a light scratchpad note widget operating entirely inside IndexedDB.
  • • Command Palette (Ctrl+K): Built-in fuzzy search across open tabs, saved bookmarks, and custom dashboard items.

Security & Google Policy Compliance

  • • Manifest V3 Enforced: Uses Service Workers for background routines with no remote code execution (eval-free).
  • • Zero Telemetry: Stripped all remote network analytics, background tracking pixels, and cloud data calls.
  • • Default OFF Privacy Options: Advanced modules (Draft Recovery & Local Stats) default to OFF per Chrome Developer Guidelines.

Want to request a feature for the next version? Submit your ideas on our GitHub Issue Tracker.
